# First ML Project

A beginner-friendly end-to-end Machine Learning project built using Python and Jupyter Notebook.  
This project demonstrates the complete ML workflow including:

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Prediction

The goal of this project is to understand how real-world ML pipelines are built and structured.

---

# Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
